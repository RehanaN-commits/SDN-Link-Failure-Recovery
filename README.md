# SDN-Link-Failure-Recovery
# SDN Link Failure Detection and Recovery

## Problem Statement

Detect link failures and dynamically update routing using SDN.

## Tools Used

* Mininet
* Ryu Controller
* OpenFlow

## Setup

* Install Mininet
* Run topology using Mininet
* Run Ryu controller

## Project Goal

To ensure continuous communication by detecting failures and rerouting traffic.
